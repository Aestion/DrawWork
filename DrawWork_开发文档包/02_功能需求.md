# DrawWork 功能需求文档

## 一、画板（Board）

### 1.1 功能描述

画板是最高层级的组织单位，相当于一个项目或工作空间。

### 1.2 功能列表

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建画板 | 用户可创建多个画板，每个画板独立 | P0 |
| 自定义命名 | 画板名称支持自定义，最多50字符 | P0 |
| 画板描述 | 可选描述信息，最多200字符 | P1 |
| 封面设置 | 从画布内容截取或上传图片作为封面 | P1 |
| 公开/私密 | 设置画板是否对内网所有人可见 | P0 |
| 删除画板 | 删除画板及所有关联数据 | P0 |
| 画板列表 | 以卡片形式展示所有画板，显示封面和名称 | P0 |
| 最近访问 | 记录并显示最近访问的画板 | P1 |
| 搜索筛选 | 按名称搜索画板 | P2 |

### 1.3 界面原型

```
┌──────────────────────────────────────────────────────────────┐
│  我的画板                                    [+ 新建画板]     │
├──────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐          │
│  │ [封面图片]  │  │ [封面图片]  │  │ [封面图片]  │          │
│  │             │  │             │  │             │          │
│  ├─────────────┤  ├─────────────┤  ├─────────────┤          │
│  │ 产品原型图  │  │ 架构设计图  │  │ 会议记录    │          │
│  │ 3个画布    │  │ 5个画布    │  │ 2个画布    │          │
│  │ [分享] [⋮]│  │ [分享] [⋮]│  │ [分享] [⋮]│          │
│  └─────────────┘  └─────────────┘  └─────────────┘          │
└──────────────────────────────────────────────────────────────┘
```

### 1.4 新建画板流程

1. 点击"新建画板"按钮
2. 弹出创建弹窗：
   - 输入画板名称（必填）
   - 输入描述（可选）
   - 选择是否公开（默认私密）
3. 确认创建
4. 系统自动创建默认画布"画布1"
5. 进入编辑器

## 二、画布（Canvas）

### 2.1 功能描述

画布是画板内的页面，一个画板可包含多个画布，类似 PPT 的多页或 Excel 的多 Sheet。

### 2.2 功能列表

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 多画布支持 | 每个画板可创建多个画布 | P0 |
| 自定义命名 | 画布名称支持自定义，默认"画布N" | P0 |
| 拖拽排序 | 支持拖拽调整画布顺序 | P1 |
| 画布切换 | 点击标签页快速切换画布 | P0 |
| 新建画布 | 添加新画布到当前画板 | P0 |
| 删除画布 | 删除画布（至少保留一个） | P0 |
| 独立内容 | 每个画布内容独立存储，互不影响 | P0 |
| 实时协作 | 每个画布独立协作，切换时保持连接 | P0 |

### 2.3 界面原型

```
┌──────────────────────────────────────────────────────────────┐
│  返回 ← 产品原型图                                           │
├──────────────────────────────────────────────────────────────┤
│  [画布 1] [画布 2] [用户调研] [架构图] [+]           [分享]  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                       [画布内容区域]                          │
│                                                              │
│                                                              │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│  [手绘工具栏]          3人在线  [导出] [设置]                │
└──────────────────────────────────────────────────────────────┘
```

### 2.4 画布标签页交互

- **单击**：切换到该画布
- **双击**：进入编辑模式，可修改名称
- **拖拽**：调整画布顺序
- **右键菜单**：重命名、删除、复制

### 2.5 画布类型

画布支持四种类型，创建时选择：

| 类型 | 说明 | 对应工具 |
|------|------|----------|
| **excalidraw** | 默认类型，手绘风格白板 | Excalidraw |
| **mindmap** | 思维导图画布 | 结构化工具 |
| **kanban** | 看板画布 | 结构化工具 |
| **swimlane** | 泳道图画布 | 结构化工具 |

**创建流程**：
1. 点击画布标签栏的 `[+]` 按钮
2. 弹出选择菜单：手绘 / 思维导图 / 看板 / 泳道图
3. 选择类型后创建对应画布
4. 画布标签显示类型图标以区分

## 三、分享功能

### 3.1 功能描述

分享以画板为单位，被分享者可以看到画板下的所有画布。

### 3.2 权限设计

| 权限 | 说明 |
|------|------|
| **所有者** | 画板创建者，拥有所有权限，可删除画板 |
| **编辑者** | 可编辑所有画布内容，可添加/删除画布，不可删除画板 |
| **评论者** | 可查看内容，可发表评论，不可编辑 |
| **查看者** | 仅可查看，不可编辑或评论 |

### 3.3 分享方式

#### 方式一：邀请用户

1. 输入用户名或邮箱
2. 选择权限级别
3. 发送邀请
4. 被邀请者收到通知，同意后获得访问权限

#### 方式二：分享链接

1. 生成分享链接
2. 选择权限级别（查看者/编辑者）
3. 可选设置：
   - 过期时间（7天/30天/永久）
   - 使用次数限制
4. 复制链接发送给他人
5. 可随时撤销链接

### 3.4 界面原型

```
┌─────────────────────────────────────────┐
│  分享画板                           [×] │
├─────────────────────────────────────────┤
│  邀请协作者                             │
│  ┌───────────────────────────────────┐ │
│  │ 输入用户名或邮箱                   │ │
│  └───────────────────────────────────┘ │
│  [查看者▼] [邀请]                      │
│                                         │
│  已邀请：                               │
│  ● 张三 - 编辑者                    [移除]│
│  ● 李四 - 查看者                    [移除]│
├─────────────────────────────────────────┤
│  分享链接                               │
│  [生成查看链接] [生成编辑链接]          │
│                                         │
│  有效链接：                             │
│  ┌───────────────────────────────────┐ │
│  │ https://drawwork.local/s/abc123   │ │
│  └───────────────────────────────────┘ │
│  权限：查看者  过期：2026-05-02      [撤销]│
└─────────────────────────────────────────┘
```

### 4.2 媒体文件支持

| 类型 | 格式 | 行为 | 实现方式 |
|------|------|------|----------|
| **静态图片** | PNG, JPG, JPEG, WEBP, SVG | 正常显示 | Excalidraw Image 元素 |
| **动态GIF** | GIF | **自动循环播放** | HTML img 标签覆盖层 |

**上传流程**：
1. 选择文件
2. 上传到 Minio 存储
3. 获取内网访问 URL
4. 在画布中显示
5. 文件信息同步到所有协作者

### 5.1 思维导图（Mind Map）

#### 功能列表

| 功能 | 描述 | 优先级 |

| 样式定制 | 节点颜色、边框样式 | P1 |
| 折叠展开 | 收起/展开子分支 | P1 |
| 导出 PNG/SVG | 导出为图片 | P2 |

#### Markdown 导入导出

支持与标准 Markdown 格式互相转换，方便在其他编辑器中使用。

**Markdown 格式规范：**

```markdown
# 中心主题
## 子主题1
- 节点A
- 节点B
  - 嵌套节点B1
  - 嵌套节点B2
## 子主题2
- 节点C
- 节点D
```

**层级规则：**
- `#` → 中心主题（根节点，仅一个）
- `##` → 一级子节点
- `-` → 子节点，通过缩进表示层级（2个空格 = 1级嵌套）
- 支持无限层级嵌套

**导入流程：**
1. 用户选择 Markdown 文件或粘贴文本
2. 系统解析层级结构（`#` = 一级，`##` = 二级，`-` = 子节点）
3. 自动生成思维导图节点树

**导出流程：**
1. 用户点击"导出 Markdown"
2. 遍历节点树，按层级输出标题和列表
3. 下载 `.md` 文件

## 六、实时协作功能

### 6.1 实时同步

多人同时编辑同一画布，实时同步所有操作。

| 功能 | 说明 |
|------|------|
| 实时同步 | 操作后 100ms 内同步给所有用户 |
| 用户光标 | 显示其他用户光标位置和名字 |
| 在线列表 | 显示当前在线用户头像 |
| 权限控制 | 根据权限决定是否可编辑 |
| 冲突解决 | Yjs CRDT 自动合并，无冲突 |
| 离线支持 | 断网时本地编辑，恢复后自动同步 |

### 6.3 评论系统

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 添加评论 | 在画布任意位置添加评论锚点 | P0 |
| 回复评论 | 支持评论下的回复（线程式讨论） | P0 |

### 6.5 计时器

| 功能 | 描述 | 优先级 |

## 九、非功能需求

### 9.1 性能要求

| 指标 | 要求 |
|------|------|
| 首屏加载 | < 3秒 |
| 画板切换 | < 500ms |
| 画布切换 | < 300ms |
| 协作同步延迟 | < 100ms（内网） |
| 支持并发 | 50人同时编辑 |
| 浏览器兼容 | Chrome, Edge, Firefox 最新版 |

### 9.2 数据安全

- 所有数据存储于内网服务器
- 定期自动备份（每日）
- 支持手动备份和恢复
- 操作日志记录

### 9.3 可用性

- Docker 一键部署
- 一键备份/恢复
- 在线更新（不停机）
- 健康检查和告警
