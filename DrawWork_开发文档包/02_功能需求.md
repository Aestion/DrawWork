# DrawWork 功能需求文档（现状对照）

> 本文档对照实际代码库更新，标记已实现/未实现的功能。
> ✅ = 已实现  |  ❌ = 未实现  |  ⚠️ = 部分实现

## 一、画板（Board）

### 1.1 功能列表

| 功能 | 描述 | 优先级 | 状态 |
|------|------|--------|------|
| 创建画板 | 用户可创建多个画板 | P0 | ✅ |
| 自定义命名 | 画板名称支持自定义，最多50字符 | P0 | ✅ |
| 画板描述 | 可选描述信息，最多200字符 | P1 | ✅ |
| 封面设置 | 从画布内容截取或上传图片作为封面 | P1 | ❌ |
| 公开/私密 | 设置画板是否对内网所有人可见 | P0 | ✅ |
| 删除画板 | 软删除画板（is_deleted 标记） | P0 | ✅ |
| 画板列表 | 以卡片形式展示所有画板 | P0 | ✅ |
| 最近访问 | 记录并显示最近访问的画板 | P1 | ✅ |
| 搜索筛选 | 按名称搜索画板 | P2 | ❌ |

## 四、画布内容功能

### 4.1 手绘功能

基于 Excalidraw 引擎，支持选择/矩形/菱形/椭圆/箭头/线条/铅笔/文本/图片等工具。✅

### 4.2 媒体文件支持

| 类型 | 行为 | 实现方式 | 状态 |
|------|------|----------|------|
| 静态图片 | 正常显示 | Excalidraw Image 元素 + 压缩 | ✅ |
| GIF | 自动循环播放 | HTML img + RAF 覆盖层 | ✅ |
| 视频 | 自动循环，静音 | HTML video 覆盖层 | ✅ |
| 音频 | 点击播放 | HTML audio 自定义控件 | ✅ |

### 4.3 文件限制

| 类型 | 最大大小 | 状态 |
|------|----------|------|
| 图片 | 1.5MB（自动压缩） | ✅（compressCanvas） |
| 视频 | 无硬限制（MIME 验证） | ✅ |

### 4.4 导出功能

| 格式 | 说明 | 状态 |
|------|------|------|
| PNG | 高清位图导出 | ✅（Excalidraw 原生 + html-to-image） |
| SVG | 矢量图导出 | ✅（Excalidraw 原生） |
| JSON | Excalidraw 原生格式 | ✅ |
| Markdown | 思维导图导出 | ✅（mindmap-utils） |
