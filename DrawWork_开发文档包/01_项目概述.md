# DrawWork 项目概述

## 项目简介

DrawWork 是一款面向公司内部使用的在线协作白板工具，基于 Excalidraw 引擎开发，支持手绘风格、富媒体内容、结构化工具（思维导图/泳道图/看板）和实时多人协作（评论/投票/计时器）。

## 核心功能

| 模块 | 功能描述 |
|------|----------|
| **画板** | 用户可创建多个画板，自定义命名，支持封面设置 |
| **画布** | 每个画板可包含多个画布，自定义命名，支持拖拽排序 |
| **分享** | 以画板为单位分享，权限分为编辑者/查看者/评论者 |
| **手绘** | Excalidraw 原生手绘功能，支持多种图形和样式 |
| **媒体** | 支持图片、GIF（自动循环）、视频（自动循环）、音频（点击播放） |
| **思维导图** | 支持创建思维导图，支持 Markdown 导入导出 |
| **泳道图** | 支持水平/垂直泳道，适合流程分析 |
| **看板** | 支持多列任务管理，拖拽移动卡片 |
| **评论** | 在画布任意位置添加评论，支持回复和 @ 提及 |
| **投票** | 支持对画布元素发起投票，实时计票 |
| **计时器** | 主持人控制倒计时，全员可见，适合工作坊场景 |
| **协作** | 多人实时协作，同步显示其他用户光标和操作 |
| **通知** | 站内通知系统，分享邀请、评论回复、投票结果实时推送 |

## 部署方式

**内网部署**：部署于公司内网服务器，数据完全物理隔离，保障企业数据安全。

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | React + Excalidraw | 18 / 0.17.6 |
| 后端 | Node.js + Express | 20 LTS |
| 实时协作 | Yjs + y-websocket | 最新版 |
| 数据库 | PostgreSQL | 15 |
| 缓存 | Redis | 7 |
| 文件存储 | Minio | 最新版 |
| 反向代理 | Nginx | Alpine |
| 容器化 | Docker + Docker Compose | - |

## 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/your-company/drawwork.git
cd drawwork

# 2. 配置环境变量
cp config/.env.example config/.env
# 编辑 config/.env，设置必要参数

# 3. 一键部署
./scripts/deploy.sh

# 4. 访问
# http://服务器IP 或 http://drawwork.company.local
```
