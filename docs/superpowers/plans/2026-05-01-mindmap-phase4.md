# 思维导图 Phase 4: Markdown 导入/导出实现计划

## 目标
完善 Markdown 导入/导出功能，支持多根节点和媒体占位符。

## 设计要点

### 导出格式
```markdown
# 思维导图

## 根节点1
1. **子节点1**
   - [图片:cat.gif]
   - [连接到:根节点2的子节点A]
   1. **孙节点1**
   2. **孙节点2**
2. **子节点2**

## 根节点2
1. **子节点A**
   - [图片:dog.jpg]
```

### 导入规则
- 一级标题 `##` 视为根节点
- 有序列表 `1.` 或无序列表 `-` 视为子节点
- 缩进决定层级
- `[图片:xxx]` / `[视频:xxx]` 占位符静默忽略
- `[连接到:xxx]` 占位符静默忽略

### 格式示例

**导出示例：**
```markdown
# 思维导图

## 项目规划
1. **需求分析**
   - [图片:requirements.png]
   1. **用户调研**
   2. **竞品分析**
2. **技术方案**
   - [连接到:架构设计]

## 架构设计
1. **前端架构**
2. **后端架构**
```

**导入支持：**
- `## 标题` → 根节点
- `1. **文本**` / `- **文本**` → 普通节点
- `- [图片:xxx]` / `- [视频:xxx]` → 忽略（占位符）
- `- [连接到:xxx]` → 忽略（占位符）
