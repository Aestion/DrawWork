# 思维导图 Phase 1: 多中心架构实现计划

## 目标
将现有单根节点结构重构为多根节点结构，支持创建、删除、布局多个独立的思维导图树。

- [ ] 修改 calculateTreeLayout 支持多树布局
- [ ] 更新 save/load API 调用使用新格式

- [ ] 可以创建多个根节点

- [ ] 树与树之间自动保持间距
- [ ] 删除根节点时子树被级联删除
- [ ] 数据正确保存和加载
