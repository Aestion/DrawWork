## 14. 常见问题排查

### Q1: Devtools 无法连接到 Chrome
**解决**:
- 检查 Chrome 是否已安装
- 检查 `--user-data-dir` 目录是否存在且有写权限
- 尝试删除用户数据目录后重试

### Q2: 页面加载超时
**解决**:
- 确认后端服务已启动
- 检查端口 3000, 3001, 5173 是否被占用
- 增加等待时间

### Q3: 协作测试失败
**解决**:
- 确认 Yjs 服务运行正常
- 检查两个 devtools 实例使用不同的用户数据目录
- 验证 WebSocket 连接没有被防火墙阻挡
