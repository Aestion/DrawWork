# MindMap Editor 功能测试计划

> 范围：MindMapEditor 组件 + useMindMapYjs hook 所有交互功能
> 环境：http://localhost:5173，登录后创建一个思维导图画布

---

## 1. 基础编辑

| # | 测试 | 步骤 | 预期结果 |
|---|------|------|---------|
| 1.1 | 创建画布 | 左侧面板 → + → 选择"思维导图" | 画布创建成功，显示默认根节点 |
| 1.2 | 编辑节点文字 | 双击根节点 → 输入文字 → 回车或点击空白 | 文字更新 |
| 1.3 | 创建子节点 | 选中节点 → 按 Tab | 在下方创建子节点 |
| 1.4 | 创建兄弟节点 | 选中节点 → 按 Enter | 在同级位置创建兄弟节点 |
| 1.5 | 删除节点 | 选中节点 → 按 Delete → 确认 | 节点及其子树被删除 |
| 1.6 | 多根节点 | 选中节点 → Ctrl+Enter | 创建新的独立根节点（最多 10 个） |

## 4. 跨树连接

| # | 测试 | 步骤 | 预期结果 |
|---|------|------|---------|
